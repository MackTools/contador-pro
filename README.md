# 📊 Contador Pro - Sistema de Gestión Contable

Aplicación contable con sincronización a la nube.

## Características
- ✅ Libro diario, balanza y cuentas T
- ✅ Fórmulas tipo Excel (ej: =A1+B2*100)
- ✅ Cálculos automáticos de debe/haber
- ✅ Exportación a Excel y PDF
- ✅ Sincronización con MongoDB Atlas
- ✅ Acceso desde web y escritorio

## Instalación
```bash
pip install -r requirements.txt
python main_ui.py
